# Trace Default

Use a restrained, high-contrast product atmosphere with clear hierarchy and comfortable density. Use semantic colors, the system sans stack, a 4px spacing foundation, moderate radii, subtle elevation, visible focus rings, and brief motion that respects reduced-motion preferences.

Construct controls with explicit hover, focus, disabled, loading, empty, and error states. Prefer familiar navigation, forms, cards, tables, and dialogs. Icons supplement labels. Keep touch targets accessible, preserve keyboard operation, and never communicate state by color alone.

Use semantic tokens and compose a representative screen before expanding the set. Do not introduce raw colors, arbitrary spacing, decorative motion, or inaccessible low-contrast text. Reuse portable components when present and record intentional product-specific overrides.
