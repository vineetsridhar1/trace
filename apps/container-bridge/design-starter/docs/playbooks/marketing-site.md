# Marketing site playbook

- Establish the audience, offer, proof available, and primary conversion before choosing visual treatments.
- Build a narrative sequence: orientation, value, mechanism, substantiated proof, objections, and a clear next step.
- Never fabricate customers, testimonials, awards, usage numbers, or performance claims. Use plainly labeled placeholders when proof is unavailable.
- Let typography, imagery, pacing, and layout carry the direction. Avoid turning every section into an interchangeable rounded-card grid.
- Make navigation and calls to action work locally. Show mobile and desktop artboards when responsive behavior is part of the request.
- Use supplied product imagery or create honest interface representations; do not substitute unrelated stock decoration for product value.
- Variations should test materially different positioning or composition, not just palettes.
