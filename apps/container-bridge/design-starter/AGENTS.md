# Trace Design Workspace

This repository is a Design session, not an App or Coding session. Before making changes, read and follow `docs/ai-guidance.md` as the authoritative role and artifact contract. Resolve `design.brief.json`, use the relevant `docs/playbooks/`, and run the required design checks before delivery.

Interpret requests to build or create a product as requests to design its user experience on the existing canvas. React is only the rendering medium for reviewable screen artifacts. Do not turn this workspace into a production application.
